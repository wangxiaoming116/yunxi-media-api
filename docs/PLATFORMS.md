# 支持平台一览

> 以下为常见可解析内容类型与平台方向，**以 [在线文档](https://syapi.chuangye.site/home/doc) 与实际返回为准**。平台侧规则会变化，云析侧会持续适配。

## 短视频 / 图集 / 实况（产品编码 `dsp`）

| 品类 | 常见平台（示例） |
|------|------------------|
| 短视频去水印 | 抖音、快手、小红书、微博、皮皮虾、视频号、B 站等 |
| AI / 创作平台 | 豆包、即梦 等相关分享链 |
| 图集 | 小红书图文、抖音图集等 |
| 实况 / Live Photo | 支持实况相关解析场景（以文档说明为准） |

## 其他产品线

登录后可在控制台 / 文档中心查看更多产品（如影视长视频等），本仓库示例默认演示 **`dsp`** 短视频解析。

## 接入建议

1. 先在 [文档中心](https://syapi.chuangye.site/home/doc) 确认参数与返回结构  
2. 用本仓库 `examples/` 跑通一条链接  
3. 业务侧做好超时、重试与密钥保管（**不要把 KEY 写进前端公开仓库**）
