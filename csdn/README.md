## 发布时注意

1. CSDN 新建博客 → 粘贴 Markdown（或切换到 Markdown 编辑器）
2. **摘要**用文首「导语」段；封面可截官网首页
3. **标签**建议：`去水印` `API` `抖音` `快手` `小红书` `短视频` `Python` `Node.js`
4. **分类**：后端开发 / 云计算/大数据 / 程序员 等按你账号习惯选
5. 文中不要填真实 UID/KEY；示例一律 `YOUR_UID` / `YOUR_KEY`
6. 文末固定放官网与开源示例仓库链接
